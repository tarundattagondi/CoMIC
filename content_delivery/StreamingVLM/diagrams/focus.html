Design Choices

Training-free: reuses existing LLM cross-attention, no new parameters.

Per-frame only: ranks within frame, no cross-frame reweighting. Orthogonal to temporal methods (STAMP-T).

Spatial focus: reduces Nspat per frame, not frame count T.

Source layer: cross-attention extracted from LLM layer 0 prefill pass.

★ vs. Prior Work

FastV: rank by LLM self-attention; layer-local, not text-conditioned.

FasterVLM: CLS-attention; image-only, no temporal.

CDPruner: joint DPP; we only do quality, no diversity term.

PruneVid: feature cosine similarity; no query signal.

Ours: text-conditioned, per-frame, training-free, composable with STAMP-T.
